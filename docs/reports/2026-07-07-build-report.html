What happened

Three parallel reviewers swept the codebase with different lenses: correctness, Next.js/SEO/accessibility, and portfolio presentation. Every finding was implemented the same day as five stacked branches with one commit per numbered item, then a sixth request added a visual showpiece pass on the Bed Designer. Each commit passed TypeScript and ESLint; each branch passed a full production build; visual work was screenshot-verified in a real browser; dataset changes were checked by invariant scripts (companion symmetry, archetype coverage, note coverage). The stack merged to main as a clean fast-forward and went live the same evening.

Shipped, by area

Data & correctness

  • Fixed: Hawaii / Puerto Rico / Guam ZIPs (zones 12–13) always saw the "no crop matches" empty state; tropical zones now share the zone-11 crop list
  • Fixed: zone lookup blocked up to 1.2s on the network despite an instant offline answer; the fallback now renders immediately and the API refines in the background
  • Fixed: the water-wise collection's 11 plants (okra, tepary bean, amaranth, sunflower, melon, sweet potato, and 5 herbs) existed in the archetype map and icon set but never in the dataset; restored with symmetric companion links
  • Fixed: planting-calendar months clamped at Jan/Dec, collapsing warm-zone windows to "Jan – Jan"; months now wrap the year, and fall-planted garlic shifts earlier in cold zones instead of later
  • Fixed: opening a plant from My Garden dropped the saved zone and nagged for a ZIP the app already knew
  • Fixed: the bed's "cramped" verdict band was geometrically unreachable; bands recalibrated to real packing densities
  • Fixed: four smaller items: cross-tab storage clears now sync, bed size inputs no longer fight multi-digit typing, plant lookup is O(1), and El Paso's ZIP prefix maps to its real zone

  • Plants sprout in with a staggered pop and glide to new spots when the layout reflows

  • Season scrubber: drag from day 0 to harvest and watch every circle grow at its own real rate, with ghost rings for full-grown footprints and golden rings when a plant is ready to pick
